# New Hampshire Real Estate ML Project

## Overview
This project focuses on analyzing real estate data from New Hampshire and building machine learning models to better understand housing patterns and predict property prices.

## Goals
- Explore and clean the dataset
- Perform professional exploratory data analysis (EDA)
- Build and compare machine learning models
- Identify the key factors that influence housing prices
- Document the full workflow in a clean and reproducible way

## Dataset
Source: Kaggle - New Hampshire Real Estate Data 2026

## Project Status
This project is currently in progress. The repository will be updated step by step as the workflow develops, including data cleaning, feature engineering, modeling, evaluation, and insights.

## Planned Structure
- `data/raw/` → original dataset files
- `notebooks/` → EDA and experimentation notebooks
- `src/` → reusable Python scripts for preprocessing, training, and evaluation
- `models/` → saved trained models
- `reports/figures/` → exported charts and visual assets
